# bemuse-official-server-admin-tools

GUI to help making server maintainance easier.

## Requirements

**Mac OS X Required**

- Audacity
- SoX
- bms-renderer

## Dev

```
$ yarn
```

### Run

```
$ npm start
```

### Build

```
$ npm run build
```

Builds the app for macOS, Linux, and Windows, using [electron-packager](https://github.com/electron-userland/electron-packager).
